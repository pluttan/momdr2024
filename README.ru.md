# MomDR2024

**Квест-игра ко дню рождения на ESP8266**


</div>

Игра-квест ко дню рождения, запущенная на точке доступа ESP8266. Игроки подключаются к WiFi-сети, вводят числовые коды (напечатанные внутри дартс-шариков) в браузере и следуют маршрутам с загадками через физические локации в доме и саду. Каждая цепочка маршрута заканчивается скрытым подарком.

## ■ Возможности

- ❖ **Собственная точка доступа** — ESP8266 создаёт свою WiFi-сеть (`HappyBirsday`), интернет и роутер не нужны
- ❖ **16 локаций** — 11 призовых маршрутов, 4 локации-ловушки и место для отдыха, всё раздаётся из flash-памяти
- ❖ **Навигация по кодам** — каждый шаг — уникальный 5-значный код; при вводе происходит редирект к следующей загадке
- ❖ **Подсказки-загадки** — зашифрованные подсказки (на русском) с указанием на физические объекты в реальных местах
- ❖ **Локации-ловушки** — задачи на сбор всего (7 вертушек, 20 деталей теплицы, 4 шланга, секретное задание), замедляющие игрока через счётчики прогресса в RAM
- ❖ **Раскрытие подарков** — каждый завершённый маршрут заканчивается описанием подарка
- ❖ **Интерфейс в стиле Catppuccin** — встроенный HTML с анимированным фоновым градиентом из 11 точек и карточкой входа в стиле glassmorphism

## ■ Стек

<div align="center">

| Компонент | Технология |
|-----------|------------|
| MCU | ESP8266 (NodeMCU 1.0) |
| Framework | Arduino |
| Веб-сервер | ESP8266WebServer (port 80) |
| WiFi | SoftAP mode (`WiFi.softAP`, standalone) |
| UI | Inline HTML/CSS served from sketch (Catppuccin palette, animated gradient, glassmorphism) |
| Клиентская логика | JS для каждого кода генерируется на лету по `/js?cd=<code>` для валидации ввода и редиректа |

</div>

## ■ Как это работает

```
1. ESP8266 загружается в режиме SoftAP, создавая WiFi-сеть `HappyBirsday`
2. Игрок подключает телефон к этой сети и открывает http://192.168.4.1/ — устройство раздаёт страницу входа из flash-памяти
3. В каждой физической локации есть дартс-шарик с уникальным 5-значным кодом внутри; ввод кода перенаправляет к следующей странице с загадкой
4. Текст загадки указывает на следующее место в реальном мире; игрок идёт туда и находит следующий шарик
5. Локации-ловушки требуют собрать несколько предметов (прогресс отслеживается в RAM) прежде чем маршрут продолжится
6. Завершение полной цепочки маршрута раскрывает описание спрятанного подарка на финальной странице
```

## ■ Использование

```bash
# Открыть Arduino IDE
# Выбрать плату: NodeMCU 1.0 (ESP-12E Module)
# Открыть MomDR2024.ino (routes.ino и resourses.ino компилируются вместе с ним)
# Загрузить прошивку
```

1. Включить ESP8266
2. Подключить телефон к WiFi `HappyBirsday` (пароль: `30072024`)
3. Открыть `http://192.168.4.1/`
4. Следовать загадкам и вводить коды с дартс-шариков

## ■ Локации квеста

Маршруты 01–11 — призовые цепочки; 12–15 — локации-ловушки; 16 — место для отдыха. Шаги обозначают количество экранов в каждой цепочке.

| 01 | Второй этаж | 8 | Лампа |
| 02 | Первый этаж | 15 | Батарейки |
| 03 | Старая кухня | 8 | Книга |
| 04 | Старый сад | 10 | Провод |
| 05 | Новый сад | 8 | Книга |
| 06 | Баня | 11 | Блок питания |
| 07 | Душ | 10 | Книга (Маринина) |
| 08 | Сарай | 8 | Задание: пройти малый круг |
| 09 | Теплица | 5 | Кола |
| 10 | Машина | 10 | Виски |
| 11 | Новая кухня | 8 | — |
| 12 | Вертушки (ловушка) | собрать 7 | разблокировать следующий |
| 13 | Старая теплица (ловушка) | собрать 20 | разблокировать следующий |

## ■ Файлы

- `MomDR2024.ino` — основная прошивка: запускает SoftAP и веб-сервер
- `routes.ino` — регистрирует все обработчики `/<code>`, названия локаций, текст загадок и подарки
- `resourses.ino` — строители страниц и встроенные шаблоны CSS/JS (тема Catppuccin)
